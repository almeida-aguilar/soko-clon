# SokoClon

A [Sokoban](https://en.wikipedia.org/wiki/Sokoban) clone made in C.

Rendering is handled by [Raylib](https://www.raylib.com/).

Level generation is a port of [AlliBalliBaba's Sokoban Level Generator]
(<https://github.com/AlliBalliBaba/Sokoban-Level-Generator>) from JS to C.
